Contributing the Accessibility Object Model specification

dmazzoni
2016-09-20

With this message we’d like to officially contribute the Accessibility Object Model specification, explainer, and all associated material at https://github.com/a11y-api/ to the WICG and subject it to the W3C Community Contributor License Agreement.

This spec complements existing declarative web accessibility APIs with a script-based alternative, offering improved developer productivity and filling in critical gaps in the web platform today. It’s inspired by previous proposals and has the support of the browser accessibility development teams of Google, Apple, and Mozilla. Microsoft has offered valuable feedback but is not currently participating.

If you’d like to learn more, join Alice and James during Wednesday’s TPAC Breakout Sessions in Lisbon. https://www.w3.org/wiki/TPAC2016/SessionIdeas#Accessibility_Object_Model

Alexander Surkov, Mozilla Alice Boxhall, Google Dominic Mazzoni, Google James Craig, Apple
