Donating the eCSStender engine to WICG

AaronGustafson
2016-05-18

I’d be interested in donating the eCSStender JS engine if folks think it could be valuable to the community. I built it to allow easy extension of CSS via new selectors, properties, values, and at-rules. It was also quite useful for polyfilling/prolyfilling. I wrote the bulk of it back in ’07-09 so it’s due for an update (which should make it even faster than it already is), but it could be pretty useful for quickly prototyping new CSS features in JS in order to get a feel for syntax and edge cases.

  • API: http://ecsstender.org/documentation/api/
  • Docs: http://ecsstender.org/documentation/
