eInvoice Standard Format

collimarco
2019-10-22

Currently online invoices are produced as PDF or using national formats (e.g. FatturaPA in Italy).

This makes automation and interoperability between invoicing software and SaaS extremely difficult or impossible.

Having a standard format for eInvoces would be a first step towards global automation in this field with huge economical advantages.
