Extending CSS to layout pages on a hexagonal (not cartesian) grid

Rohit-Khitchdee
2021-11-06

The html format was designed to make documents easily exchange-able over the web. All these documents were laid out on a rectangular cartesian grid. I propose we add an extension to CSS to layout a document on a hexagonal grid. Hex-grids have advantages over cartesian grids that are well documented in the literature.
