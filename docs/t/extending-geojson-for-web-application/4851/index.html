Extending GeoJSON for Web Application

AmeliaBR
2020-09-28

A breakout session at the Maps for the Web workshop looked at two potential extensions to the GeoJSON data format, to make it work with more complex applications. Bryan Haberberger talked about the effort to merge GeoJSON with JSON-LD 1.1, while Karl Grossner talked about GeoJSON-T, adding a time dimension (“when” attribute)
