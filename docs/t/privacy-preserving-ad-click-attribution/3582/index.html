Per cwilso’s request, this is to let folks on Discourse know about a proposal to allow attribution of ad clicks on the web while preserving user privacy. It’s on GitHub here:

GitHub

WICG/ad-click-attribution

Privacy Preserving Ad Click Attribution. Contribute to WICG/ad-click-attribution development by creating an account on GitHub.

Folks from WebKit, Gecko, and the NYT have expressed interest, and a number of other people have filed issues in our issue tracker. If you’re interested in contributing to this effort, please head over to the repo.
