[Proposal] Crash/Deprecation/Intervention Reporting

Ian_Clelland
2019-12-10

At TPAC in September, the W3C WebPerf WG discussed the reporting spec, and suggested that we should extract the three concrete report types from that spec (Deprecation, Intervention, and Crash reports) and set up an incubation repo for each of them. (See GitHub discussion)

I’d like to propose that we set up three repositories at WICG: crash-reporting, intervention-reporting, and deprecation-reporting, to facilitate discussion on the three topics.

cwilso
2019-12-13

Based on the Github discussion in the WebPerf WG, I’ve created these three repos: https://github.com/WICG/crash-reporting, https://github.com/WICG/intervention-reporting, https://github.com/WICG/deprecation-reporting
