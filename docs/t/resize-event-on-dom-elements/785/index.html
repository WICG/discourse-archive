Implemented in Chrome Dev 54 behind experimental web platform features flag.

new ResizeObserver(callback).observe(element)

https://crbug.com/612962

Transfering ResizeObserver to the WICG APIs
ResizeObserver is an API for observing changes to elements size. Here is an explaner: https://github.com/atotic/ResizeObserver/blob/master/explainer.md Most of the discussions of the API have happened inside Google. We’d like to open the discussion to the outside. To facilitate that, we would like to transfer ResizeObserver repo to the WICG.
