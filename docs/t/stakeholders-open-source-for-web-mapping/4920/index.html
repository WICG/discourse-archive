Stakeholders: Open Source for Web Mapping

PeterR
2020-10-03

At the joint W3C-OGC Maps for the Web workshop, we held a panel of open source stakeholders in standards for Web mapping, including browser stakeholders, the open source Web map developer community and open source user organizations.

Panel members included: @zcorpan Simon Pieters, of Bocoup, Daniel Morissette of OSGEO/MapGears, @ahocevar Andreas Hocevar of OpenLayers and Will Mortensen of NGA Gnome program. The panel was moderated by Tom Kralidis of OSGEO.
