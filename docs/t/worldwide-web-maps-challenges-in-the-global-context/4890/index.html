Worldwide Web Maps: Challenges in the global context

PeterR
2020-10-01

@Shepazu Doug Schepers moderated a panel about challenges relating to standardization, or lack thereof, of maps on the Web. Panelists included Brandon Liu of ProtoMaps, Siva Pidaparthi, of ESRI, and Thijs Brentjens, of Geonovum (NL).

Super interesting topics of the discussion included:

  • Spatial data on the Web best practices
  • localization, multi-lingual maps, and synergies between what is already natively implemented by browsers for internationalization support versus what must be done “manually” by JavaScript libraries for Web maps
  • Security and privacy of personal geospatial data in the international context
  • Map projections and the challenges of sophisticated mathematics (e.g. turf.js) associated to even simple Web maps
  • Accessibility and Web mapping in the international context
