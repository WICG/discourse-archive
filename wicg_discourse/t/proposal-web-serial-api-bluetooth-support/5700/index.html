[Proposal] Web Serial API Bluetooth support

cmumford
2021-12-02

Hi WICG:

Bluetooth Low Energy and mesh are very popular, but BR/EDR (AKA “classic”) still has active development and ships on most Bluetooth devices today. Support for the Serial Port Profile (SPP) routinely comes up in developer requests. There is no good equivalent to SPP in BLE, so we propose adding support for Bluetooth classic serial ports to the Web Serial API.

We’ve put together an explainer, and would love to hear everyone’s feedback on this proposal.
