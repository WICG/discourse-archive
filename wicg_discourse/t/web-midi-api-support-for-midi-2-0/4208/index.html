Web MIDI API support for MIDI 2.0

rektide
2020-01-30

Greetings all. MIDI 2.0 was approved on January 19th, 2020. I’d like to see Web MIDI API updated or complemented with support for MIDI 2.0.

There’s a lot of new capabilities in 2.0! It’ll probably take some work to get there, but it’ll be great when the Web can fully make use the newly expanded Musical Instrument Digital Interface.

Just a stub right now, but I did open a repository for this initiative. I’ll look for better source/reference material soon to start work off of.

I also opened web-midi-api#211 to discuss this, and followed the steps to file a Project Fugu feature request with Chromium.
